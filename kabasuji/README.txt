Group Poseidon 
Authors: Monika Danielewicz, Morgan Hopeman, Natalia Kononenko, Alex Titus, and Jacob Wennersten

This is the first implementation of the Kabasuji game. 

HOW TO RUN
To run the Level Builder application:
1. Open the src folder 
2. Open the poseidon.builder.veiw package
3. Run the Main.java file

To run the Level Player application:
1. Open the src folder
2. Open the poseidon.player.view package
3. Run the Main.java file
